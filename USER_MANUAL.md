# AI陪伴机器人平台系统使用手册

## 1. 产品概述

### 1.1 产品简介

AI陪伴机器人平台是一款基于AI技术的智能机器人管理平台，支持设备管理、动作库管理、AI对话交互等功能，帮助用户轻松管理和控制机器人设备。

### 1.2 核心功能

- **用户管理**：注册、登录、个人信息管理
- **设备管理**：设备绑定、状态监控、设备列表管理
- **动作库管理**：预设动作、自定义动作、动作组合
- **AI对话**：自然语言交互、指令识别、动作匹配
- **数据分析**：设备运行数据、动作执行统计
- **个性化设置**：主题切换、布局自定义

### 1.3 适用角色

- **普通用户**：使用平台管理机器人设备
- **管理员**：管理用户、设备、权限等
- **开发人员**：对接API、扩展功能

## 2. 快速入门

### 2.1 注册与登录

1. **注册账号**：
   - 访问平台首页
   - 点击「注册」按钮
   - 填写用户名、邮箱、密码
   - 点击「注册」完成注册

2. **登录平台**：
   - 访问平台首页
   - 点击「登录」按钮
   - 选择登录方式（邮箱/手机号）
   - 输入登录信息
   - 点击「登录」进入平台

### 2.2 绑定设备

1. 登录平台后，进入「设备管理」页面
2. 点击「添加设备」按钮
3. 填写设备名称和设备类型
4. 点击「添加」完成设备绑定

### 2.3 执行动作

1. 在设备列表中选择一个设备
2. 进入设备详情页面
3. 在动作库中选择一个动作
4. 点击「执行」按钮，机器人将执行对应动作

## 3. 功能模块使用指南

### 3.1 用户管理

#### 3.1.1 个人信息管理

1. 点击右上角头像，选择「个人设置」
2. 可修改昵称、邮箱、手机号等信息
3. 点击「保存」完成修改

#### 3.1.2 密码修改

1. 进入「个人设置」页面
2. 点击「修改密码」
3. 输入原密码和新密码
4. 点击「保存」完成密码修改

#### 3.1.3 多因素认证

1. 进入「安全设置」页面
2. 开启「多因素认证」
3. 扫描二维码，使用认证APP添加
4. 输入验证码完成绑定

### 3.2 设备管理

#### 3.2.1 设备列表

- 查看所有绑定的设备
- 显示设备名称、类型、状态
- 支持设备搜索和筛选

#### 3.2.2 设备详情

- 设备基本信息
- 设备状态实时监控
- 动作执行历史
- 设备日志

#### 3.2.3 设备操作

- **添加设备**：绑定新设备
- **编辑设备**：修改设备信息
- **删除设备**：解绑设备
- **查看详情**：查看设备详细信息

### 3.3 动作库管理

#### 3.3.1 预设动作

- 平台提供的基本动作
- 不可修改，只能使用
- 包括前进、后退、左转、右转等

#### 3.3.2 自定义动作

1. 进入「动作库」页面
2. 点击「添加动作」按钮
3. 填写动作名称、描述、时长
4. 配置动作步骤
5. 点击「保存」完成创建

#### 3.3.3 动作组合

1. 进入「动作库」页面
2. 点击「创建组合动作」
3. 选择多个动作
4. 设置动作执行顺序和间隔
5. 点击「保存」完成创建

### 3.4 AI对话

1. 进入设备详情页面
2. 在对话输入框中输入指令
3. 点击「发送」按钮
4. AI将识别指令并匹配对应动作
5. 机器人执行动作并返回结果

### 3.5 数据分析

1. 进入「数据分析」页面
2. 选择时间范围和图表类型
3. 查看设备运行数据
4. 查看动作执行统计
5. 下载数据报表

### 3.6 个性化设置

1. 进入「设置」页面
2. **主题切换**：选择浅色/深色主题
3. **布局设置**：选择垂直/水平/混合布局
4. **快捷操作**：配置常用功能快捷键
5. 点击「保存」应用设置

## 5. 开发人员指南

### 5.1 API文档

- API文档地址：`http://localhost:3000/api-docs`
- 支持OpenAPI 3.0规范
- 提供在线调试功能
- 包含请求响应示例

### 5.2 快速开始

1. 获取API密钥
2. 阅读API文档
3. 调用API接口
4. 处理响应数据

### 5.3 常见接口示例

- **用户登录**：`POST /api/auth/login`
- **获取设备列表**：`GET /api/devices`
- **执行动作**：`POST /api/devices/:id/actions`
- **创建动作**：`POST /api/actions`
- **AI对话**：`POST /api/chat`

## 6. 常见问题解答

### 6.1 登录问题

**Q：忘记密码怎么办？**
A：在登录页面点击「忘记密码」，按照提示重置密码。

**Q：登录失败怎么办？**
A：检查用户名/密码是否正确，确认账号状态是否正常。

### 6.2 设备问题

**Q：设备无法绑定怎么办？**
A：检查设备网络连接，确认设备是否支持该平台。

**Q：设备离线怎么办？**
A：检查设备电源和网络连接，重启设备后重试。

### 6.3 动作执行问题

**Q：动作执行失败怎么办？**
A：检查设备状态，确认动作配置是否正确。

**Q：动作执行延迟怎么办？**
A：检查网络连接，减少并发操作。

### 6.4 其他问题

**Q：如何切换主题？**
A：进入「设置」页面，选择主题选项。

**Q：如何联系客服？**
A：通过平台内「帮助中心」联系客服。
